# ft_nm

## Symbol List

|Item|Description|
|----|-----------|
|A|	Global absolute symbol.|
|a|	Local absolute symbol.|
|B|	Global bss symbol.|
|b|	Local bss symbol.|
|D|	Global data symbol.|
|d|	Local data symbol.|
|f|	Source file name symbol.|
|L|	Global thread-local symbol (TLS).|
|l|	Static thread-local symbol (TLS).|
|T|	Global text symbol.|
|t|	Local text symbol.|
|U|	Undefined symbol.|

## Usage

```shell
$> ./ft_nm [options] [file ...]
```
